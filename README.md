🌍 Perfect Journey Creator – Travel Agency Website

This repository contains the source and deployment link for my travel agency website created using AI tools. The website is designed to provide users with a smooth and user-friendly experience for exploring travel packages, including Northern Pakistan tours, Umrah, and Hajj services.

📌 Project Overview

Perfect Journey Creator is a modern travel agency website that allows users to:

Explore tour packages for popular destinations like Skardu, Swat, and Kashmir
Get information about Umrah and Hajj travel services
Navigate easily through a clean and responsive interface
Experience a professional layout suitable for a real business
